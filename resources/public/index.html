To convert SQL to DSQL, use a POST request to the /to-dsql endpoint.

Request Example:

POST /to-dsql
Content-Type: application/json

{
    "sql": "SELECT * FROM table1 WHERE element > $1",
    "params": [30]
}

Parameters:

  • Request body: SQL query as a plain string.
  • params (optional): parameters for substitution, if used.

Response:

The response body will contain the DSQL expression or an error message.
